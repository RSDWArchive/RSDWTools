How do I use this?

Click on the plus (+) icon in the Player Inventory section and select your Save file.

Make changes, add stuff, remove stuff, manually set the quantity of an item to 1,000,000... then hit Save!


Your Save file is here (paste this into your file explorer):

%localappdata%\RSDragonwilds\Saved\SaveCharacters\


Disclaimer: data is compiled from the latest game files automatically. Always back up your save before editing.
